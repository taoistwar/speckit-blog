使用 speckit 生成的博客网站，仅用于测试

#### Step 1: 创建章程 (Constitution)

```bash
/speckit.constitution Create principles for a personal blog: simple static site generation, Markdown-based content, fast page loads, SEO-friendly, no JavaScript required for reading.
```

#### Step 2: 创建功能规格 (Specify)

```bash
/speckit.specify Build a personal blog where I can write posts in Markdown, organize them by tags, and have readers subscribe to email updates when new posts are published.
```

#### Step 3: 创建技术计划 (Plan)

```bash
/speckit.plan Use Eleventy (11ty) as static site generator, Netlify for hosting and form submission for emails. Store subscribers in a JSON file (Mailchimp integration optional). Use Tailwind CSS for styling.
```

#### Step 4: 生成任务列表 (Tasks)

```bash
/speckit.tasks
```

#### Step 5: 执行实现 (Implement)

```bash
/speckit.implement
```
