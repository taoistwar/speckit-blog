Welcome to my new personal blog!

This blog is built with [Eleventy](https://www.11ty.dev/), a simple static site generator that takes Markdown files and transforms them into beautiful HTML pages.

## Why Eleventy?

I chose Eleventy because:

1. **It's fast** - Pages are pre-rendered at build time
2. **It's flexible** - No lock-in to specific frameworks
3. **It's simple** - Configuration is straightforward
4. **It's powerful** - Great plugin ecosystem

## What I'll Be Writing About

Expect posts on:

- Web development and JavaScript
- Programming best practices
- Tools and workflows
- Occasional life updates

## Getting Started

To create a new post, simply add a Markdown file to the `src/posts/` directory with the following frontmatter:

```yaml
---
title: "Your Post Title"
date: 2026-04-17
tags:
  - your-tag
---

Your content here...
```

Thanks for reading, and stay tuned for more content!
